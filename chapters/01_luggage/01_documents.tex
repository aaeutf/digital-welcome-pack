\subsection{证件类}

\textbf{必备证件：}

\begin{itemize}
    \item 护照（含签证）
    \item 出生证明及双认证原件、翻译件、复印件
    \item 国内驾照原件、公证件、复印件（如有）
    \item 学校录取信（含学校地址与邮箱信息）
    \item 毕业证、学位证、成绩单复印件及翻译件
    \item 证件照若干（纸质及电子版）
    \item 在法住宿信息（地址、电话等）
    \item 个人简历（英文或法文版）
\end{itemize}

\textbf{建议：}

\begin{itemize}
    \item 所有证件建议扫描保存电子档，并至少准备2份备用复印件，分别放在不同行李中。
    \item 每个行李最好放一份机票和护照复印件，以备航空公司行李延误或丢失时方便找寻。
    \item 保留好登机牌和行李牌。
\end{itemize}
